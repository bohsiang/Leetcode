### Q :
> Given an integer, return its base 7 string representation.

> ```
> Input: 100
> Output: "202"
> ```

***

### A:



```python
class Solution:
    def convertToBase7(self, num: int) -> str:
        n ,ans =abs(num), "" 
        if n == 0: ans = "0" 
        while n:
            ans = str(n % 7) + ans
            n //= 7
        
        return ans if num >= 0 else "-" + ans
        
 ```
